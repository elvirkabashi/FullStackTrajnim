Bëhu pjesë e trajnimit FullStack

Mirë se vini në trajnimin tonë FullStack në ASP .NET Core Web API dhe React JS! Ky kurs ofron një eksperiencë të plotë për të mësuar të zhvilloni aplikacione web të avancuara duke përdorur teknologjitë më të fundit dhe më të kërkuara në industrinë e zhvillimit të softuerit.

Në këtë kurs, do të shkoni në një udhëtim të thellë në botën e ASP .NET Core Web API dhe React JS, duke filluar nga themelimi i një projekt i gatshëm dhe duke përfunduar në krijimin e aplikacioneve web të shkallës së gjërë që janë të sigurtë, të shpejtë dhe të përdorshëm.

Në pjesën e ASP .NET Core Web API, do të mësoni si të ndërtoni dhe të shpërndani API-të web të fuqishme duke përdorur infrastrukturën moderne të .NET Core. Do të zgjeroni njohuritë tuaja për mënyrat më të mira për të modeluar dhe për të trajtuar kërkesat e klientit.

Në pjesën e React JS, do të eksploroni një nga kornizat më të preferuara të zhvillimit të aplikacioneve web. Do të mësoni si të ndërtoni komponente të rindërtueshme dhe të përshtatshme, si të menaxhoni gjendjen e aplikacionit me saktësi duke përdorur 'state' dhe 'props', si dhe si të lidhni komponentët me API-të e përmirësuara të cilat keni zhvilluar në pjesën ASP .NET Core Web API të kursit tonë.

Në fund të kursit, do të jeni në gjendje të kuptoni dhe të ndërtoni aplikacione web të plota FullStack duke përdorur teknologjitë ASP .NET Core Web API dhe React JS. Përgatitu për të shkuar përtej nivelit themelor dhe për të zgjeruar aftësitë tuaja në botën e zhvillimit të aplikacioneve web të avancuara

Git & GitHub

  • Çfarë është sistemi kontrolli i versioneve dhe pse është i rëndësishëm për zhvillimin e softuerit.
  • Përparësitë e përdorimit të Git për menaxhimin e kodit dhe bashkëpunimin në një projekt.
  • Si të krijoni një repositor në GitHub dhe si të kryeni operacione të zakonshme si commit, push, pull dhe merge.

Crud Operations

  • Kuptimi dhe aplikimi i operacioneve CRUD (Create, Read, Update, Delete) në bazën e të dhënave.
  • Implementimi i operacioneve CRUD në një aplikacion të shërbimit web ASP .NET Core.

Web API

  • Çfarë është një API web dhe si funksionon.
  • Zhvillimi i API-ve web të fuqishme në ASP .NET Core, duke përfshirë modelimin e burimeve, ruajtjen e të dhënave dhe implementimin e endpoint-ve të API-ve.

API Calls in React

  • Përdorimi i teknologjive të përparuara të klientit si React JS për të krijuar aplikacione web.
  • Si të kryeni kërkesa HTTP dhe të konsumoni të dhëna nga API-të e zhvilluara në pjesën e shërbimit web ASP .NET Core.
  • Demonstrimi i komunikimit midis frontend dhe backend për të shfaqur dhe manipuluar të dhënat nëpërmjet një ndërfaqeje të përdoruesit.

Auth & JWT Token

  • Kuptimi i autentifikimit dhe autorizimit në aplikacionet web.
  • Implementimi i sistemeve të autentifikimit në ASP.NET Core, përfshirë shfletuesin dhe shërbimin e serverit.
  • Implementimi i JWT (JSON Web Token) për të siguruar dhe për të menaxhuar sesionet e përdoruesit.
  • Zhvillimi i logjikës për të verifikuar dhe për të ruajtur tokenin JWT për përdoruesit e autentikuar.
  • Implementimi i autorizimit bazuar në role për të kufizuar qasjen në resurse dhe veprimet e aplikacionit në bazë të rolit të përdoruesit.

Planprogrami i zgjeruar i trajnimit

Në vijim gjeni planprogramin e zgjeruar të trajnimit.

  • Krijimi i repositorit të GitHub
  • Git & Visual Studio Code
  • Përgatitja e ambientit të punës (Visual Studio)
  • Testimi i Api default me Swagger UI
  • Lidhja me MSSQL (Connection String)
  • Instalimi i paketës Entity Framework
  • Krijimi i kontekstit të bazës së të dhënave dhe konfigurimi i Entity Framework për lidhjen me databazën
  • Krijimi i modelit të parë
  • Shtimi i `DbSet` për secilën tabelë në modelin e databazës përkatëse
  • Migrimi i modelit dhe shfaqja në bazën e të dhënave
  • Controller-i dhe Action Metodat për Operacionin CRUD
  • Linjat (routes)
  • Testimi i CRUD në API me Swagger UI ose Postman

  • Përgatitja e ambientit të punës
  • Si krijohet një projekt
  • Komponentat
  • JSX
  • React props
  • Destrukturimi i vetive (destructuring props object)
  • Validimi i vetive të komponenteve
  • Renderimi me kusht (conditional rendering)
  • Eventet
  • Menaxhimi i gjendjes (State Managment)
  • Dërgimi i gjendjes dhe funksioneve tek komponentet fëmijë përmes props
  • Linjat (routes) / React router
  • UseEffect
  • Konsumimi i APIs (axios)
